# App Icon Design Guide - Homelyo Professional

## Brand Identity

**App Name**: Homelyo Pro
**Tagline**: Professional Service Partner
**Brand Color**: #6366F1 (Indigo)
**Theme**: Professional, Trustworthy, Modern

## Design Concept

### Primary Icon Concept: Professional Badge
A clean, modern icon representing professional service excellence.

**Visual Elements**:
- **Main Symbol**: Toolbox or wrench icon (representing service professionals)
- **Background**: Gradient from #6366F1 (indigo) to #4F46E5 (darker indigo)
- **Accent**: White symbol with subtle shadow for depth
- **Style**: Flat design with slight gradient, rounded corners

### Alternative Concepts

**Option 2: House + Tools**
- Minimalist house outline with tools icon overlay
- Represents home services

**Option 3: Letter "H" Badge**
- Stylized "H" for Homelyo
- Professional badge shape
- Tool elements integrated into letter

## Technical Specifications

### App Icon (iOS & Android)
- **Size**: 1024x1024 px (master file)
- **Format**: PNG with transparency
- **Color Mode**: RGB
- **DPI**: 72 (screen resolution)
- **Safe Zone**: Keep important elements within 90% of canvas (avoid edges)
- **No Text**: Icon should be recognizable without text

### Adaptive Icon (Android Only)
- **Size**: 1024x1024 px
- **Foreground**: Icon elements (with transparency)
- **Background**: Solid color #6366F1
- **Safe Zone**: Center 66% circle (edges may be cropped)

### Notification Icon (Android Only)
- **Size**: 96x96 px
- **Style**: White icon on transparent background
- **Complexity**: Simple, recognizable at small sizes
- **Format**: PNG with alpha channel

### Splash Screen
- **Size**: 1242x2436 px (iPhone 13 Pro Max)
- **Design**: Simple logo on brand background
- **Background**: #6366F1 solid or gradient
- **Logo**: Centered, max 40% of screen height
- **Style**: Minimal, fast-loading feel

## Design Guidelines

### DO:
✅ Use brand color #6366F1 as primary
✅ Keep design simple and recognizable
✅ Ensure readability at 40x40px (smallest display size)
✅ Use consistent visual style across all assets
✅ Test on both light and dark backgrounds
✅ Include subtle shadows for depth
✅ Use rounded corners (iOS style)

### DON'T:
❌ Use gradients that are too complex
❌ Include small text or fine details
❌ Use photos or realistic imagery
❌ Copy competitor designs
❌ Use more than 3 colors
❌ Make icon too busy or cluttered

## Figma Template Structure

If creating in Figma:

1. **Canvas Setup**:
   - Create 1024x1024 artboard
   - Add grid with 8px spacing
   - Create safe zone guide (90% circle)

2. **Layers**:
   - Background (gradient rectangle)
   - Icon symbol (vector shape)
   - Shadow effect (subtle)
   - Export frame

3. **Export Settings**:
   - PNG format
   - Transparent background for foreground layer
   - Multiple scales (1x, 2x, 3x)

## Testing Your Icon

### Checklist:
- [ ] Recognizable at 40x40px
- [ ] Looks good on white background
- [ ] Looks good on black background
- [ ] Stands out among other apps
- [ ] Matches brand identity
- [ ] No copyright issues
- [ ] Scales well to all sizes
- [ ] Passes app store guidelines

### Preview On:
- iPhone home screen (various models)
- Android home screen (various launchers)
- App Store listing
- Play Store listing
- Push notification badge (small)
